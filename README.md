# JWT express example

## Features

* Babel & ES2015
* Mongoose
* Mocha for tests
* ESlint

## Install
```
npm install
```

## Scripts

#### Start the project for development
```
npm run dev
```

#### Run All tests
```
npm test
```

#### Run ESlint
```
npm run lint
```

#### debug
```
node-inspector  # lauch this in a new tab
npm run debug

netstat -p tcp | $port # find if some app is using defualt port:5858
ps ax | grep node      # find process id to Kill #number
 ```
